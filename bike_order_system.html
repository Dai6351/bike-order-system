このプロトタイプの現在地と、本番化に必要な追加要素
現在のプロトタイプ構成（フロントエンドのみ）
ユーザー

フロント
HTML / CSS / JavaScript単一ファイル（SPA）

データ
localStorage（ブラウザ内）リロードで消えない ※同端末のみ

↓
バックエンド

将来AI拡張
Claude API / OpenAI API商品提案・問い合わせ自動応答
DBテーブル設計（Supabase想定）
orders（客注テーブル）
id UUID PRIMARY KEY
order_no TEXT （例: KJ-12345）
product_name TEXT
maker TEXT
customer_name TEXT
customer_tel TEXT
member_id TEXT （会員の場合）
status TEXT （waiting / ordered / arrived / done）
note TEXT
created_at TIMESTAMP
updated_at TIMESTAMP

products（商品マスタ）
id UUID PRIMARY KEY
name TEXT
maker TEXT
price INTEGER
stock_status TEXT （in / low / out）

members（会員マスタ）
member_id TEXT PRIMARY KEY
name TEXT
tel TEXT
rank TEXT
Vercelデプロイ手順（3ステップ）
1
GitHubにHTMLをpush
github.comでリポジトリを作成 → このHTMLファイルをアップロード
2
Vercelでインポート
vercel.com → 「Add New Project」→ GitHubリポジトリを選択 → Deploy
3
URLが発行される
https://your-project.vercel.app のURLで即日公開。無料。
